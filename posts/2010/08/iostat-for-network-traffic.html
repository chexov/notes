iostat for network traffic

August 2 2010, 5:19 PM  by Anton Checkhov

This shows the amount of traffic on each network interface as well as any network errors,
dropped packets, and overruns. This also shows the name of the network interface (for example, eth0).

watch command helps to see network usage over time:
watch --interval 0.5 netstat -i inet


To identify which processes are using the network, you can use lsof.
The -i4 parameter shows which processes have IPv4 connections,
-i6 displays IPv6, -i tcp lists TCP, and -i udpdisplays applications with open UDP sockets
