netstat, process PID and OS X

December 30 2008, 4:55 PM  by Anton Checkhov

How to do something like the Linux "netstat -naptu" on the Mac:

lsof -nPi
Explanation:
-i lists IP connections (both IPv4 and IPv6). -n Numbers only, for the IP addresses (prevents a lot of name resolution, which is annoying if you can't access a DNS server. -P Port numbers also only in numeric form. Else, they would be converted to service names using /etc/services.
