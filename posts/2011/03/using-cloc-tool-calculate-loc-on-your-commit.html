Using cloc tool to calculate LOC of any git commit

March 25 2011, 7:52 PM  by Anton Checkhov

In case you want to know you Lines Of Code counters for any git commit use awsome insructions below.

You need cloc tool to get it working right.

 

Put these lines into your ~/.gitconfig:

[difftool "cloc"]
 cmd = cloc --diff $LOCAL $REMOTE

[alias]
 cloc = difftool -t cloc -y

If you already have [alias] section, append that to the end of it.
